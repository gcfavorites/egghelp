# Название #
cleanusers.tcl


# Автор #
<sup>Неизвестен</sup>


# Описание #
Скрипт проверяет раз в день юзерфайл и удаляет из него тех юзеров, которых не видел в течении указанного числа дней. Также может удалять тех юзеров, у которых не установлен пароль.
