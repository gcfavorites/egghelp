# Название #
mainserver.tcl


# Автор #
<sup>Неизвестен</sup>


# Описание #
Скрипт старается держать бота только на одном сервере, при том, что в конфиге указано несколько. Необходимо в том случае, когда вы хотите чтобы все боты вашего ботнета использовали разные серверы.
